# .dotfiles

1. Link dotfiles: `./link.sh`
2. Install basic tools: `./Brewfile`
3. Add `DEFAULT_USER` envvar to `~/.zshrc_local`: `export DEFAULT_USER='manuel'`
4. Use Configure iTerm to use [Fira Code](https://github.com/tonsky/FiraCode)
